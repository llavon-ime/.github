# 拉風輸入法

拉風輸入法（Llavon IME）是一套以現代大型語言模型(LLM)技術棧為核心打造的繁體中文注音輸入法。

不同於主要依賴固定詞頻、靜態詞庫與人工規則的傳統輸入法，拉風輸入法會綜合前文語境、注音序列與已選文字進行預測，提供更符合上下文的選字結果。

## 特色

- **理解上下文**  
  根據前後文判斷相同讀音在不同語境下的合適用字，減少逐字修正的需要。

- **專為注音輸入訓練**  
  使用針對繁體中文注音輸入打造的語言模型，不是將一般聊天模型直接接到傳統輸入法上。

- **兼顧語意與讀音**  
  模型負責理解語境，預測結果則受到合法注音候選集合約束，在改善選字能力的同時維持讀音正確性。

- **完全本機推論**  
  輸入內容不需要傳送到雲端，能在本機完成候選字預測，兼顧隱私與離線使用需求。

- **輕量化模型**  
  採用約 2.5 億參數的 LLaMA 架構模型，並以 GGUF 量化格式執行，在模型能力與日常裝置效能之間取得平衡。

- **跨平台架構**  
  核心推論服務與輸入法前端彼此分離，讓不同平台能共用模型、詞表與推論邏輯，也便於替換模型及持續改善使用體驗。

- **開放原始碼**  
  輸入法程式、推論服務與相關工具皆以開放協作為方向發展。

## 運作方式

拉風輸入法會將使用者輸入的注音、目前已選文字與前文語境交由專用語言模型分析，再從符合讀音的候選集合中選出最適合的結果。

模型透過 `llama.cpp` 在本機執行，支援 CPU 推論，並可在環境允許時使用 GPU offload。前端只負責接收輸入與呈現候選字，模型載入及推論則由共用的背景服務處理。

## 專案目標

我們希望打造一套：

- 真正理解繁體中文語境的注音輸入法
- 不依賴雲端服務也能提供智慧選字的輸入工具
- 能在多種桌面環境中提供一致核心體驗的開放專案
- 讓模型、輸入法前端與訓練方法都能持續演進的實驗平台

## 開發狀態

拉風輸入法目前仍在積極開發中，模型、功能、操作方式與安裝流程都可能持續調整。
